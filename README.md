# 💰 Expense Tracker (Full Stack Python Project)

## 📌 Overview

Expense Tracker is a simple and efficient full-stack web application built using Python.
It helps users record, manage, and analyze their daily expenses with a clean and interactive UI.

---

## 🚀 Features

* ➕ Add new expenses
* 📊 View all expenses in a table
* 🔍 Filter expenses by category
* 🗑 Delete individual expense
* ❌ Clear all expenses
* 💾 Persistent storage using SQLite
* 📈 Real-time total expense calculation

---

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **Libraries:** Pandas

---

## 📁 Project Structure

```
expense_tracker/

│── requirements.txt
│── README.md
```

---

## ⚙️ Installation

### 1. Clone the repository (or create project manually)

```
git clone <your-repo-url>
cd expense_tracker
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Running the Application

```
streamlit run app.py
```

The app will open in your browser at:

```
http://localhost:8501
```

---

## 🧪 Example Usage

1. Enter expense title
2. Enter amount
3. Select category
4. Click **Add**
